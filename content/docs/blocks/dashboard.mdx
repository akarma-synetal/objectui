## Usage

Customize dashboard blocks for your needs:

### Add Real Data

```json
{
  "type": "card",
  "dataSource": {
    "api": "/api/stats/revenue",
    "method": "GET"
  }
}
```

### Customize Colors

```json
{
  "type": "text",
  "content": "+20.1%",
  "className": "text-emerald-500"
}
```

### Add Actions

```json
{
  "type": "button",
  "label": "View Details",
  "action": {
    "type": "navigate",
    "path": "/dashboard/details"
  }
}
```
