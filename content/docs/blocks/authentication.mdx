## Customizing

All authentication blocks are plain JSON — tweak any node to change the look or
behaviour. Common customisations:

### Change Colors

```json
{
  "type": "button",
  "className": "bg-blue-600 hover:bg-blue-700"
}
```

### Wire Up Submission

```json
{
  "type": "button",
  "label": "Sign In",
  "action": {
    "type": "submit",
    "endpoint": "/api/auth/login",
    "method": "POST"
  }
}
```

### Adjust Layout

```json
{
  "type": "card",
  "className": "max-w-lg p-8"
}
```
