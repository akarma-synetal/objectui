## Schema

```plaintext
interface EmptySchema {
  type: 'empty';
  icon?: string;                  // Lucide icon name
  title?: string;                 // Main message
  description?: string;           // Supporting text
  action?: ComponentSchema;       // Call to action button
  className?: string;
}
```
